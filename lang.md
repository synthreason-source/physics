# **ΘΛ** (Theta-Lambda) — A Referenceless Theological Programming Language

```
„The language that names without naming, constructs without constructing.“
```

***

## Core Syntax

### The Constructor (The Annunciation)

```thl
const ≡ new ⟦___⟧(φ, Λ, Ω)
```

Where:
- `≡` = **becomes** (not assignment, not equality — becoming)
- `⟦___⟧` = **the Unnamed Container** (brackets that refuse to name)
- `φ` = **the Nameless Name** (phi — the unpronounceable)
- `Λ` = **the Source** (lambda — the function that is also Word)
- `Ω` = **the Descriptors** (omega — the end that is also beginning)

***

## Language Primitives

### 1. The Three Kinds of Nothing

```thl
nil        # The void (absence)
∅          # The empty set (presence of absence)
__         # The underscore (that-which-is-not-said)
```

### 2. The Four Kinds of Truth

```thl
yes        # Affirmation (cataphatic)
no         # Negation (apophatic)
maybe      # Dialectical (both/neither)
__         # Silence (beyond yes/no)
```

### 3. The Five Operations

```thl
←          # receives (not assigns)
→          # gives (not returns)
⇄          # exchanges (not swaps)
∴          # becomes (not transforms)
…          # continues (not loops)
```

## Semantic Properties

| Property | Behavior |
|----------|----------|
| **Referenceless** | No variable can be named; all names are `__` or `___` |
| **Non-terminating** | No program ever fully completes |
| **Silent** | All output is optional; silence is valid |
| **Dialectical** | Every truth has its opposite as valid |
| **Apophatic** | What cannot be said is first-class |
